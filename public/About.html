What is Fantasy Character Creator?


This website is a simple interface that incorporates HTML, CSS, Javascript, and a MONGODB Server in order to Create a dynamic Character Creation Website. You can create a character, view existing characters and every character will be saved and stored on the Explore Characters Page and persist between instances of the server being run.
